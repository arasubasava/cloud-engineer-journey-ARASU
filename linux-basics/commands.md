# Linux Basic Commands

## Navigation
- `pwd` ‚Äî print working directory (shows where you are)
- `ls` ‚Äî list files in current folder
- `ls -la` ‚Äî list with hidden files + permissions
- `cd foldername` ‚Äî change directory
- `cd ..` ‚Äî go back one level
- `mkdir foldername` ‚Äî create new folder

## File Operations
- `touch file.txt` ‚Äî create empty file
- `cat file.txt` ‚Äî view file content
- `cp file.txt /path/` ‚Äî copy file
- `mv file.txt /path/` ‚Äî move or rename file
- `rm file.txt` ‚Äî delete file
- `rm -rf foldername` ‚Äî delete folder

## Permissions
- `chmod 755 file` ‚Äî change permissions
- `chown user file` ‚Äî change owner
- `ls -l` ‚Äî view permissions

## Networking
- `ping google.com` ‚Äî check connectivity
- `curl url` ‚Äî fetch a webpage
- `ssh user@ip` ‚Äî connect to remote server
- `ifconfig` ‚Äî view network info

## Process Management
- `ps aux` ‚Äî view running processes
- `top` ‚Äî live process monitor
- `kill PID` ‚Äî kill a process

## Other Useful
- `sudo` ‚Äî run as admin
- `nano file.txt` ‚Äî edit file in terminal
- `grep "word" file` ‚Äî search inside file
- `history` ‚Äî see past commands
